Soundly-Addition
======================
このライブラリはLWJGLでのサウンド機能を強化するものです。  
このライブラリは[Soundly](http://code.google.com/p/slick-soundly/)ライブラリを改変しています。  
 
追加機能
------
* OGG&MODフォーマットのXStreamingSoundをInputStreamから生成出来るように
* コンパイル時に警告が出ないように修正

関連情報
--------
